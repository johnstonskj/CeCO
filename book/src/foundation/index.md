# Foundation

The CeCO Foundation is a domain-independent set of core concepts providing simple
but formal building blocks for the domain-specifics in the following sections. The
core is organized into the following logical sections.

- [**Core**](./core/index.md); a set of five concepts that are the the basis for
  all classes in CeCO.
- [**Identity Aspects**](./identity/index.md); fundamental classes for
  classification and identification.
- [**Structural Aspects**](./structural/index.md); aspects for describing the
  fundamental structure of things.
- [**Spatial Aspects**](./spatial/index.md); aspects for describing spatial
  things.
- [**Temporal Aspects**](./temporal/index.md); aspects for describing temporal
  events and spans.
- [**Agreements**](./agreements/index.md); the fundamental notions of agreements
  between parties.
- [**Artifacts**](./artifacts/index.md); the very basic notion of an artifact as
  a thing.

The IRI for the Ontology is [`<https://ceco.one/v/2026/06/01/foundation/>`](https://ceco.one/v/2026/06/01/foundation/).
